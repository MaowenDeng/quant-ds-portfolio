# Quantitative Finance & Data Science Portfolio

This repository serves as a **central landing page** for my selected projects in **quantitative finance and data science**.

My work focuses on applying statistical and computational methods to analyze financial data, with an emphasis on **return-risk tradeoffs, asset behavior, and interpretability**.

---

## Area of Focus
- Stock return analysis
- Portfolio construction and risk modeling
- Financial data analysis (R, Python & C++)
- Interactive data visualization (R Shiny)

Each project repository contains **full documentation, code, and results**, and is designed to be reproducible and easy to explore.
